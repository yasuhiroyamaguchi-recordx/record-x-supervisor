### 補足 4: multiple instance 並走運用(EVT-008 + 回覧板 v1.1 整合)

監督官 instance A / B が並走する場合、scheduler template を instance ごとに複製:

```powershell
# instance A 用(デフォルト、SUPERVISOR_INSTANCE_ID=A)
schtasks /create /xml "...layer0_supervisor_template.xml" /tn "Record-X-Layer0-Supervisor-A" /F

# instance B 用(XML を編集 + SUPERVISOR_INSTANCE_ID=B 設定)
schtasks /create /xml "...layer0_supervisor_template_B.xml" /tn "Record-X-Layer0-Supervisor-B" /F
```

並走時の lock 機構:
- 同一時刻起動 → 後続 instance は「Concurrent run avoided」で exit 0
- 起動時刻をずらす運用が現実的(A: 03/09/15/21、B: 06/12/18/24 等)

L1 回覧板(`internal/circular/`)経由で各 instance の cycle 終了結果が共有される。
